# DoAnIoT

## Mô tả
Dự án IoT

## Cấu trúc thư mục
- `arduino/` - Mã nguồn Arduino
- `main.py` - File Python chính

## Cách chạy

### Python
```bash
python main.py
```

### Arduino
Mở file `arduino/arduino.ino` trong Arduino IDE và upload lên board.

## Yêu cầu
- Python 3.x
- Arduino IDE (cho phần arduino)
