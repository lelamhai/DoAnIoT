# Hướng dẫn thêm ảnh người thân

## Cấu trúc
- Mỗi người 1 folder riêng
- Tên folder = tên hiển thị khi nhận diện
- Mỗi folder chứa 3-10 ảnh với nhiều góc độ và ánh sáng khác nhau

## Ví dụ:
```
known_faces/
├── Linh/
│   ├── 01.jpg
│   ├── 02.jpg
│   └── 03.jpg
└── Nam/
    ├── 01.jpg
    ├── 02.jpg
    └── 03.jpg
```

## Lưu ý:
- Ảnh nên rõ mặt, không bị che khuất
- Kích thước tùy ý, hệ thống sẽ tự resize
- Format: jpg, jpeg, png
- Nhiều ảnh = độ chính xác cao hơn
